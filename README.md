# nokuchimbetete.vercel.app

Personal portfolio for Noku Chimbetete — developer, writer, artist.
Built with Astro. Deployed free on Vercel.

## Quick start

```bash
npm install
npm run dev
# → http://localhost:4321
```

## First steps after downloading

1. Add your portrait photo → `public/images/portrait.jpg`
2. Add your hover video → `public/videos/portrait-video.mp4`
3. Add project thumbnails → `public/images/project-1.jpg`, etc.
4. Edit `src/pages/index.astro` — fill in bio and real projects
5. Update social links in `src/layouts/Base.astro`
6. Push to GitHub, connect to Vercel

See **CLAUDE.md** for full instructions (including how to work with Claude Code).
